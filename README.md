## BingWallpaperToday

Get the bing.com wallpaper of today

    go get github.com/weaming/bingwptoday
    bingwptoday -n 3

## Usage

    $ bingwptoday -h
    Usage of bingwptoday:
      -d string
            Directory to store wallpapers. (default "~/Downloads/Bing Wallpapers")
      -n string
            Number of wallpapers to download. (default "1")
